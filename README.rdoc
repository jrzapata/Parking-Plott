== README

##Urban Yoda Translator

###Description

This is a web app that let users list, browse and reserve(beta) private parking spots in NYC.

###Tech used / Stack

- Ruby

- Rails

- JavaScript

- Postgres



###Features

- User can list a private parking spot with the app.

- User can browse all available spots in the NYC areas.

- User can reserve a spot with the app.


### Future Implementations

- Better email actions.
